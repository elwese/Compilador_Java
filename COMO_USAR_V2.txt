CREAR TUS PROPIOS ARCHIVOS .java
================================

1. Crea un archivo con extensión .java:
   mi_codigo.java

2. Escribe código Java (sintaxis básica):
   int x = 10;
   float y = 3.14;
   
   if (x > 5) {
       System.out.println("Mayor que 5");
   }

3. Guarda en la misma carpeta que main.py

4. Ejecuta:
   python main.py

5. Selecciona el archivo del menú


FORMATO ESPERADO DE ARCHIVOS .java
==================================

✓ CORRECTO:
  int x = 10;
  float y = 3.14;
  
  if (x > 5) {
      x = x + 1;
  }

❌ INCORRECTO (Java completo, no soportamos):
  public class Main {
      public static void main(String[] args) {
          int x = 10;
      }
  }

(El compilador solo analiza bloques de código, no clases completas)

ERRORES COMUNES
===============

❌ "Archivo no encontrado"
   ✓ Asegúrate de escribir la ruta correcta
   ✓ Verifica que el archivo esté en la carpeta correcta
   ✓ Usa: python main.py y selecciona opción 1 si está listado

❌ "No se encuentran archivos .java"
   ✓ Crea al menos un archivo .java en la carpeta
   ✓ Usa los ejemplos: ejemplo.java, ejemplo_completo.java

❌ "Error de encoding"
   ✓ Asegúrate de que tu archivo .java esté en UTF-8
   ✓ En VS Code: Ctrl+Shift+P → "Change File Encoding" → UTF-8

TIPS ÚTILES
===========

1. Para analizar rápidamente:
   - Selecciona 0 para probar
   - Selecciona 1 para ejemplos

2. Para agregar más ejemplos:
   - Crea archivos .java en la misma carpeta
   - Aparecerán automáticamente en el menú

3. Para editar ejemplos:
   - Abre ejemplo.java o ejemplo_completo.java
   - Edita el código
   - Guarda
   - Ejecuta python main.py de nuevo

4. Para copiar la ruta de un archivo:
   - Abre el archivo en VS Code
   - Haz clic derecho → "Copy Path"
   - Usa esa ruta en la opción 2
